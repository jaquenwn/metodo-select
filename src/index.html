Cómo funciona el Ordenamiento Burbuja:

  1. Comparar por pares: Desde la primera posición compara con la siguiente y determina la mayor entre ellas
  2. Intercambiar: Si la primera carta es mayor guarda el valor temporalmente e intercambia el index con la carta menor.
  3. Repetir: Repite este proceso para cada carta por cada posicion del arreglo.
